# E-commerce Vue.js

Projeto de e-commerce desenvolvido utilizando Vue.js, vue-router e vuex.

## Tecnologias Utilizadas

- Vue.js
- Vue Router
- Vuex

## Instalação
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
